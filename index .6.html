The Road to Enlightenment

Littering a dark and dreary road lay the past relics of browser-specific tags, incompatible DOMs, broken CSS support. We must clear the mind of the past. Web enlightenment has been achieved thanks to the tireless efforts of folk like the W3C, WaSP, and the major browser creators.

The CSS Zen Garden invites you to relax and meditate on the important lessons of the masters. Begin to see with clarity. Learn to use the time-honored techniques in new and invigorating fashion. Become one with the web.

So What is This About?

There is a continuing need to show the power of CSS. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view the selected style sheets in the list. Clicking any will apply it to this page.

CSS allows complete and total control over the style of a hypertext document. The only way this can be illustrated in a way that gets people excited is by demonstrating it in real-world action. Designers and coders alike can participate.

Participation

Strong visual design has always been our focus. You are modifying this page, so strong CSS skills are necessary too, but the example files are commented and the code is clean and straightforward. Download the sample HTML and CSS and give it a try.

You may modify the style sheet in any way you wish, but not the HTML. This may seem daunting at first if you’ve never worked this way before, but follow the listed guidelines and you’ll be fine.

Download the sample HTML and CSS files to work on your own version.

Benefits

Why participate? For recognition, inspiration, and a resource we can all refer to showing people how amazing CSS really can be. This site serves as a demonstration and a learning tool.

Requirements

Where possible, we would like to see mostly CSS 1 & 2 usage. CSS3 & 4 should be handled with care and consideration for backward compatibility.

Luckily, designing this way shows how well various browsers have implemented CSS by now. When sticking to the guidelines you should see fewer inconsistencies and more harmony across browsers.

We ask that you submit original artwork. Please respect copyright laws. Please keep objectionable material to a minimum, and try to incorporate unique and interesting visual themes into your design.

This is a learning exercise as well as a demonstration. You retain full copyright on your graphics (with limited exceptions, see submission guidelines).
